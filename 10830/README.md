## 풀이 전.
> 매트릭스의 제곱 문제. </br></br>
> 매트릭스 A의 N 제곱은 이렇게 나타낼 수 있다.</br>
> A ^ N 에서 N이 짝수인 경우, 홀수인 경우로 구분. </br> </br>
> (짝수인 경우) </br>
> A ^ N = (A ^ 2) ^ (N / 2) </br></br>
> (홀수인 경우) </br>
> A ^ N = (A ^ 2) ^ ((N-1) / 2) * A</br></br>
> 이러면 N번이 아닌 log(N)번으로 계산 시간을 줄일 수 있다.
___
