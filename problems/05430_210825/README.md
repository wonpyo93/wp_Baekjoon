## 풀이 전.
> 문제는 길지만 사실상 R(뒤집기), D(버리기)가 있다.</br>
> 뒤집기는 말그대로 뒤집는거고 버리기는 말그대로 첫째 원소 버리기다.</br>
> 배열을 array로 잡거나 vector로 잡거나의 시간 차이가 이 문제 해결의 키 포인트가 될 것 같다.
